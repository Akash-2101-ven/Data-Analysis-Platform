# Data Analysis Platform

An interactive full-stack Data Analysis Platform built using React, FastAPI, Tailwind CSS, and Recharts. The platform enables users to upload datasets, generate automated KPIs, visualize insights through interactive charts, and interact with an AI-powered analytics assistant.

---

## Features

### Data Upload & Processing
- Upload CSV and Excel files
- Automated dataset parsing and schema detection
- Dynamic data preview generation

### Dashboard & Visualization
- Automated KPI generation
- Interactive Bar Chart visualization
- Pie Chart analytics
- Trend Line analytics
- Responsive dashboard interface

### AI-Powered Analytics
- AI Data Copilot for natural language interaction
- Dataset-aware contextual responses
- Automated business insight generation

### Data Exploration
- Structured data explorer table
- Dynamic schema mapping
- Interactive visualization components

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Recharts
- Axios

### Backend
- FastAPI
- Pandas
- NumPy
- OpenAI API

---

## Workflow

1. Upload a CSV or Excel dataset
2. Backend processes and analyzes the data
3. KPIs and visualizations are generated dynamically
4. Interactive charts display analytical insights
5. AI Copilot assists users with dataset-related queries
